# products-management-C-
Educational project in C#. Product inventory management system.

Учебный проект, реализованный на C#.

Функционал:
- добавление продуктов;
- удаление записей;
- поиск продуктов;
- анализ сроков хранения;
- определение наиболее скоропортящегося продукта.

Использованные технологии:
- C#
- ООП
- Работа с коллекциями данных
